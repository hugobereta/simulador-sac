# simulador-sac
Simulador de Operações Pré-Fixadas - Sistema SAC

### BB Giro Empresa ###
A metodologia utilizada baseia-se na metodologia da linha BB Giro Empresa, do Banco do Brasil S.A.

### Importante ###
Esta é uma produção totalmente independente, sem quaisquer tipo de envolvimento de qualquer Instituição Financeira,
inclusive sem validação dos cálculos pelo Banco. O intúito aqui é unica e exclusivamente a criação de uma ferramenta
que facilite os cálculos, entendimentos e possa auxiliar na tomada de decisão sobre o negócio.
